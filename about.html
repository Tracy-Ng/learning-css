Learning CSS

The objective of the CSS LEARNING work sessions is to allow students to become autonomous in the acquisition of CSS techniques and innovations. Over the last three years CSS has introduced new and exciting innovations that have established CSS as an essential discipline for any Web professional. In this hyper-dynamic and changing ecosystem, the important thing is to understand the fundamentals and to know when and how to adopt these new features. It's also about having fun.

Technologies used

  • Media queries
  • Background changes for 4 screen sizes
  • Object fit
  • Flexbox
  • CSS grid layout
  • Transforms
  • Animations
